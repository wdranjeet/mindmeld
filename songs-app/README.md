## Installation

```bash
cd songs-app
npm install
```

## Development

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```

## PWA Features

- Works offline after first visit
- Installable on mobile devices
- Caches API responses for offline playback
- Service worker for background sync

## Usage

1. Open the app in your browser
2. Browse trending Hindi bhakti songs on the homepage
3. Use the search bar to find specific songs or artists
4. Click on any song to view details
5. Install the app on your mobile device for offline access
